Overview

This page demonstrates how to structure and format future modules in the Research Paper Toolbox website. You can copy this file and use it as a starting point for a new lesson page.

Tip

Use sidebars (<aside>) for helpful notes, examples, or “Being Realistic” callouts in your modules.

Section Title Example

Each module should be divided into logical sections. Use <h3> for main section headings and <h4> for subsections.

Include figures and captions when referring to visuals or screenshots.

Subtopic Example

For secondary explanations, break the content into smaller paragraphs or lists. Example:

  • Use short, scannable sentences.
  • Include meaningful headings.
  • Keep visuals relevant and accessible.

Accessibility Reminder

All images should have clear alt attributes for screen readers. Decorative icons may use alt="".

Conclusion

Summarize the main takeaways of the lesson and direct users to the next step (such as a quiz or another module). This ensures continuity across the course.
