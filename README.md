Overview
The **Research Paper Toolbox** is an educational website designed to guide graduate students through each stage of writing and organizing research papers.  
It introduces tools, methods, and resources that help with topic selection, data formatting, research design, and professional communication.

This project was developed as part of the **COM 421: Technical Communication** course at the Illinois Institute of Technology.
